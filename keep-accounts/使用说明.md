## 1. 启动应用

### 开发模式（通过命令行）

打开终端（Terminal），依次输入以下命令：

```shell
# 第一步：进入项目目录
cd ~/Project/keep-accounts

# 第二步：启动应用
npm run dev
```

稍等几秒，应用窗口会自动弹出。

> ⚠️ **注意**: 启动命令必须在 `keep-accounts` 目录下运行，否则会报错。项目脚本已自动处理环境变量问题，无需手动设置。

### 关闭应用后重新启动

如果关闭了终端窗口或退出了应用，只需重新打开终端，重复上述两步即可。

## 3. 记账操作

点击左侧 **✏️ 记账** 进入记账页面。

### 记录一笔支出

1. 确保顶部切换按钮选中「💸 支出」（红色高亮）
2. 在「金额」输入框中输入金额，比如 `35.50`
3. 在「分类」中选择对应的分类，比如 🍽️ 餐饮
4. 日期默认是今天，可以点击修改
5. 备注可填可不填，比如"午餐外卖"
6. 点击「✓ 记录」按钮完成

### 记录一笔收入

1. 点击顶部「💰 收入」按钮（绿色高亮）
2. 后续步骤与支出相同，分类会自动切换为收入分类（工资、奖金等）

### 修改一笔交易

1. 切换到「📋 账单」页面
2. 鼠标悬停在要修改的交易上
3. 点击「编辑」按钮
4. 表单会自动填入原来的数据，修改后点击「💾 保存修改」

### 删除一笔交易

1. 切换到「📋 账单」页面
2. 鼠标悬停在要删除的交易上
3. 点击「删除」按钮
4. 再次点击「确认」完成删除，点击「取消」则放弃

## 4. 查看账单

点击左侧 **📋 账单** 进入账单页面。

### 页面布局

- **顶部三张卡片**: 显示当前筛选条件下的总支出、总收入、总结余
- **筛选栏**: 可按类型和月份筛选交易
- **交易列表**: 按日期分组，最新的在最上面

### 筛选交易

- **按类型**: 点击「📋 全部」「💸 支出」「💰 收入」切换
- **按月份**: 点击下拉框选择特定月份，或选择「全部月份」

### 交易记录说明

每条交易显示：
- 🍽️ **分类图标和名称**
- **备注内容**（如有）
- **金额**: 红色带 `-` 号是支出，绿色带 `+` 号是收入

## 5. 统计报表

点击左侧 **📊 统计** 进入统计页面。

### 月度汇总

- 页面顶部显示「◀ 上月」和「下月 ▶」按钮，可切换不同月份
- 三张卡片显示当月的**收入**、**支出**和**结余**

### 支出分类饼图

- 以饼图形式展示当月各支出分类的占比
- 右侧列出各分类的具体金额
- 如果没有支出数据，显示空状态提示

### 月度趋势柱状图

- 展示近 6 个月的收入/支出变化趋势
- 绿色柱 = 收入，红色柱 = 支出
- 便于对比每月的收支情况

## 6. 分类管理

点击左侧 **📂 管理分类** 打开分类管理弹窗。

### 查看分类

弹窗中列出所有内置分类，分为「💸 支出分类」和「💰 收入分类」两组。

### 隐藏不常用的分类

- 每个内置分类右侧有「隐藏」按钮
- 点击后该分类在记账页面不再显示
- 再次点击「显示」可恢复

> 内置分类不可删除，只能隐藏。这是为了保护数据一致性。

### 添加自定义分类

1. 点击弹窗底部的「＋ 添加自定义分类」
2. 选择类型（支出或收入）
3. 输入 emoji 图标（可直接用系统输入法输入，如输入"餐"会出现🍽️）
4. 输入分类名称（最多 10 个字）
5. 点击「添加」

自定义分类添加后立即生效，可在记账页面看到。

## 7. 数据导出

在「📋 账单」页面，筛选后的汇总栏右侧有「📥 导出」按钮。

1. 点击「📥 导出」按钮
2. 在弹出的保存对话框中，选择保存位置和文件名
3. 点击「保存」

导出的 CSV 文件包含以下列：
- 日期
- 类型（支出/收入）
- 分类
- 金额（支出为负数，收入为正数）
- 备注

CSV 文件可以直接用 **Excel**、**WPS**、**Numbers** 等软件打开，方便进一步分析。

## 8. 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌘ + N` | 快速新建交易（跳转到记账页） |
| `⌘ + 1` | 切换到记账页 |
| `⌘ + 2` | 切换到账单页 |
| `⌘ + 3` | 切换到统计页 |
| `Esc` | 取消编辑状态 |

> 💡 「⌘」是 Mac 键盘上的 Command 键（⌘）。Windows 上对应的是 Ctrl 键。

## 9. 退出应用

### 方式一：关闭窗口（推荐）

点击窗口左上角的 **红色关闭按钮**（●），或按 `⌘ + W`。

- 在 Mac 上，关闭窗口后应用图标仍保留在程序坞，可再次点击打开
- 在 Windows 上，关闭窗口即退出应用

### 方式二：完全退出

在 Mac 上，按 `⌘ + Q` 或点击菜单栏「生活财务小管家 → 退出」。

### 方式三：从终端退出

如果应用是通过 `npm run dev` 启动的，在终端中按 `Ctrl + C` 即可同时关闭应用和开发服务器。

## 10. 常见问题

### Q: 启动时报错 "Cannot read properties of undefined"

**原因**: 没有在 `keep-accounts` 目录下运行命令。

**解决**:
```shell
cd ~/Project/keep-accounts
npm run dev
```

### Q: 启动后窗口是空白的

**原因**: 开发服务器未正确启动。

**解决**: 在终端中按 `Ctrl + C` 关闭，重新运行 `npm run dev`。

### Q: 数据保存在哪里？

- **Mac**: `~/Library/Application Support/keep-accounts/data.db`
- **Windows**: `%APPDATA%/keep-accounts/data.db`

数据全部保存在你的电脑上，不会上传到任何服务器。

### Q: 如何备份数据？

复制上述路径中的 `data.db` 文件到其他位置即可。恢复时粘贴回原路径。

### Q: 如何迁移到新电脑？

1. 复制旧电脑的 `data.db` 文件
2. 在新电脑上安装并启动一次应用（会自动创建数据目录）
3. 退出应用
4. 将备份的 `data.db` 覆盖新电脑上的同名文件
